# g4nggaa.api

Multi-platform web & API project (tanpa Vercel)

## 🔧 Struktur

- `index.js` → Backend Express (deploy di Railway)
- `public/` → Frontend statis (deploy ke Netlify atau GitHub Pages)
- `src/api/` → API AI & download
- `package.json` → Konfigurasi Node.js

## 🚀 Deploy

### 1. Frontend (Netlify / GitHub Pages)
- Deploy folder `public/`

### 2. Backend (Railway)
- Deploy seluruh folder ini (pastikan `index.js` & `package.json`)
